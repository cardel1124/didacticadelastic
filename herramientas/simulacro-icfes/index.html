Simulacro ICFES

Plataforma propia para simulacros de pruebas Saber 11, con registro de estudiantes, panel de administrador y estadísticas por área.

Esta herramienta necesita Node.js corriendo en tu computador. GitHub Pages solo sirve archivos estáticos, así que esta página no puede iniciar el servidor por ti — pero sí puedes correrlo en minutos en tu propio equipo.

Cómo usarlo

  1. Instala Node.js (versión 18 o superior).
  2. Descarga o clona este repositorio.
  3. Abre una terminal en la carpeta herramientas/simulacro-icfes.
  4. Ejecuta npm install.
  5. Copia .env.example como .env y define tu propio ADMIN_PASSWORD y SESSION_SECRET.
  6. Ejecuta node server.js.
  7. Abre http://localhost:3000 en tu navegador.

Instrucciones completas, flujo de uso y escala de calificación en el README de esta carpeta.
